# 你是哪个历史人物？

基于大五人格（OCEAN）科学的性格测试网站，通过 50 道混合题型评估，匹配 16 位中国历史人物，找到跨越千年的灵魂共鸣者。

## 功能特性

- **50 题混合题型** — 30 道 Likert 自评量表 + 20 道情境选择题
- **五维人格分析** — 开放性(O)、尽责性(C)、外向性(E)、宜人性(A)、神经质(N)
- **16 位历史人物** — 诸葛亮、李白、曹操、苏轼、武则天、王阳明、康熙、项羽、刘备、孙子、鲁迅、屈原、张良、成吉思汗、林黛玉、关羽
- **欧氏距离匹配** — 科学量化你与历史人物的人格相似度
- **可视化报告** — ECharts 雷达图 + 柱状图 + 深度维度解读
- **隐私安全** — 所有数据仅存浏览器内存，不发送任何网络请求

## 快速开始

无需安装，直接双击 `index.html` 即可使用。

或用 HTTP 服务器托管：

```bash
npx serve -l 7777 -s .
# 或
python3 -m http.server 7777
```

访问 `http://localhost:7777/index.html`

## 技术栈

- 原生 HTML / CSS / JavaScript
- [ECharts 5.5](https://echarts.apache.org/) — 雷达图与柱状图
- 无构建工具、无框架、零依赖（除 ECharts CDN）

## 项目结构

```
index.html    — 页面结构（首页 / 答题 / 结果三层 section）
style.css     — 全部样式与响应式布局
script.js     — 题库、计分、匹配算法、SVG头像、图表渲染
```
